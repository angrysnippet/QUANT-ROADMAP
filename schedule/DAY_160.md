## Block 1 — The one idea that runs through everything
Every day of this module was a variation on a single theme: **no-arbitrage replication.**
- **Static** replication → put–call parity (Day 152).
- **Dynamic** replication → Black–Scholes & delta-hedging (Days 153, 155).
- **Discrete** replication → binomial trees (Day 157).
- **Simulated** replication → Monte Carlo for exotics (Day 158).
- **Static option-strip** replication → variance swaps / VIX (Day 159).

Different math, *same principle*: **build the payoff from tradeable pieces; its price is the cost of the recipe; your opinion about drift is irrelevant.** If you remember one thing from Phase 8, remember that.

## Block 2 — The pricing pipeline (your mental flowchart)
Given any derivative, you can now route it:
```
1. Write the payoff as a function of S (and the path, if any).
2. Is it path-dependent or high-dimensional?  → Monte Carlo.
3. Early exercise / American, low-dim?         → Binomial tree / PDE.
4. Vanilla European?                            → Black–Scholes closed form.
5. Price = e^{−rT} · E^Q[payoff]  (risk-neutral drift = r).
6. Differentiate for the Greeks → build the hedge → manage residual gamma/vega.
7. Quote and trade in IMPLIED VOL, not dollars.
```
That pipeline prices essentially anything you'll meet on a desk.

## Block 3 — The mental models, distilled
- **Optionality = convexity** (Jensen, Day 151): capped loss, open upside; vol has value.
- **μ disappears** (Day 153): hedging removes risk, so price under Q with drift `r`. Q ≠ P.
- **N(d₂) = ITM probability, N(d₁) = delta** (Day 153–154).
- **Long gamma ⇔ short theta** (`Θ ≈ −½σ²S²Γ`, Day 154): you pay time-decay rent for convexity.
- **Hedged option P&L ∝ σ_realized² − σ_implied²** (Day 155): options are vol bets.
- **The smile = the risk-neutral distribution** (Breeden–Litzenberger, Day 156): fat tails & crash fear, priced.
- **Pricing = DP on a tree / expectation by simulation** (Days 157–158): discretize and recurse.
- **Vol is an asset** with skew, term structure, and a risk premium (Day 159).

## Block 4 — Mathematics · the unifying equation set
Everything compresses to a handful of relations worth knowing cold:
```
Put–call parity:   C + K e^{−rT} = P + S
Risk-neutral price: V = e^{−rT} E^Q[payoff],   dS = rS dt + σS dW^Q
Black–Scholes:     C = S N(d₁) − K e^{−rT} N(d₂)
BS PDE in Greeks:  Θ + ½σ²S²Γ + rSΔ − rV = 0
Hedged P&L:        ½ ∫ Γ_t S_t² (σ_r² − σ_i²) dt
Risk-neutral density: q(K) = e^{rT} ∂²C/∂K²
```
If you can derive *parity* by replication, *recover the meaning* of `N(d₁)/N(d₂)`, and *state the hedged-P&L variance law*, you can reconstruct the rest under interview pressure.

## Block 5 — Quant Thinking · interview rehearsal (say these out loud)
Rapid-fire — answer each in 2–3 sentences before reading on:
1. *Why is a more volatile stock's call worth more, even with the same expected price?* → convexity/Jensen; the call keeps upside, caps downside at 0, so dispersion only helps (Day 151).
2. *Why isn't the expected return μ in Black–Scholes?* → dynamic hedging removes the risk; price under the risk-neutral measure where drift = r; Q is a pricing device, not a forecast (Day 153).
3. *You're long a delta-hedged straddle and nothing moves — P&L?* → lose theta; you only profit if realized vol > the implied you paid (Days 154–155).
4. *Why do OTM puts trade at higher IV?* → the market prices fat left tails / crash risk beyond log-normal; the skew premium is earned insurance, not free money (Days 156, 150).
5. *How would you price an American put? A barrier option?* → tree/PDE with `max(intrinsic, continuation)` for the American; Monte Carlo for the path-dependent barrier (Days 157–158).
6. *How do you trade volatility itself?* → variance swaps / VIX products; long vol as a crash hedge, short vol to harvest the VRP with insurance-style sizing (Day 159).

**Hard puzzle (capstone):** a client wants \$0 upfront cost but downside protection below \$90 on a stock at \$100, and is willing to cap gains above \$115 to pay for it. Design it, and explain the hedge. (A **zero-cost collar**: *buy* a 90-strike put (protection) and *sell* a 115-strike call (finance it), strikes chosen so premiums net to ≈0 — pure put–call-parity thinking (Day 152). As the dealer you're now short the put / long the call package; you **delta-hedge** the residual and manage the gamma/vega, earning the bid/offer and any skew edge. This single structure uses payoffs, parity, Greeks, and hedging together — exactly the synthesis this module built.)

## Block 6 — Python (quant) · A unified mini-pricing library
```python
import numpy as np
from scipy.stats import norm

class Pricer:
    def __init__(self, S,K,r,sig,T): self.S,self.K,self.r,self.sig,self.T = S,K,r,sig,T
    def _d(self):
        d1=(np.log(self.S/self.K)+(self.r+0.5*self.sig**2)*self.T)/(self.sig*np.sqrt(self.T))
        return d1, d1-self.sig*np.sqrt(self.T)
    def bs_call(self):
        d1,d2=self._d(); return self.S*norm.cdf(d1)-self.K*np.exp(-self.r*self.T)*norm.cdf(d2)
    def delta(self): return norm.cdf(self._d()[0])
    def mc_call(self, n=1_000_000, rng=np.random.default_rng(0)):
        ST=self.S*np.exp((self.r-0.5*self.sig**2)*self.T+self.sig*np.sqrt(self.T)*rng.standard_normal(n))
        return np.exp(-self.r*self.T)*np.maximum(ST-self.K,0).mean()
    def parity_put(self):                      # synthetic put via parity
        return self.bs_call()-self.S+self.K*np.exp(-self.r*self.T)

p = Pricer(100,100,0.05,0.2,1.0)
print(f"BS call      = {p.bs_call():.4f}")
print(f"MC call      = {p.mc_call():.4f}   (matches BS — risk-neutral expectation)")
print(f"delta=N(d1)  = {p.delta():.3f}")
print(f"put (parity) = {p.parity_put():.4f}")
```
Closed form, simulation, and parity all agree — the whole module in twenty lines.
