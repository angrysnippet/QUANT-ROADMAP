# Day 146 — 💹🔗 Finance & Portfolio World · Correlation & Diversification — The Only Free Lunch

> Combine two risky assets and — if they don't move together — the *portfolio* is less risky than either alone, with no loss of expected return. That's diversification: the closest thing to a free lunch finance offers.

## Block 1 — Covariance and correlation
- **Covariance:** `Cov(X,Y) = E[(X−μₓ)(Y−μ_y)]` — do they deviate from their means together? Units are messy.
- **Correlation:** `ρ = Cov(X,Y)/(σₓσ_y) ∈ [−1, 1]` — the standardized, unitless version.

`ρ = +1` lockstep, `0` unrelated (linearly), `−1` perfect opposite. **Correlation only captures *linear* co-movement** — two variables can be strongly dependent with ρ ≈ 0 (Day 123). In markets, correlations are unstable and *rise toward 1 in crises* (Block 4).

---

## Block 2 — The portfolio-variance formula (the heart of it)
For weights `w₁, w₂` on two assets:
```
σ²_p = w₁²σ₁² + w₂²σ₂² + 2·w₁w₂·ρ·σ₁σ₂
```
The magic is the **cross term**. Expected return combines *linearly* (`μ_p = w₁μ₁ + w₂μ₂`), but risk combines **sub-linearly whenever ρ < 1**. So you keep the average return but *shed* risk. The lower ρ, the bigger the gift; at ρ = −1 you can construct a *risk-free* combination.

**Concept check:** two assets, each σ = 20%, equal weight. Portfolio σ if ρ=1? if ρ=0? if ρ=−1? (`20%`, `√(2·0.5²·0.2²)=14.1%`, `0%`.) Same expected return in all three cases — only the risk changed.

---

## Block 3 — How diversification scales (the √N intuition)
Equal-weight `N` assets each with vol σ and *average* pairwise correlation ρ̄:
```
σ²_p = σ²/N + (1 − 1/N)·ρ̄·σ²    →    as N→∞,  σ_p → σ·√ρ̄
```
Two profound facts:
1. The *idiosyncratic* (per-asset) risk fades like `1/N` — **diversifiable risk**.
2. The `ρ̄·σ²` floor does **not** vanish — that's **systematic / market risk** you cannot diversify away (this is exactly what CAPM, Day 148, prices). If ρ̄ = 0.3, σ = 20%, the floor is `0.2·√0.3 ≈ 11%` no matter how many names you hold.

---

## Block 4 — Correlation breakdown: diversification fails when you need it most
Correlations are not constants — they are **regime-dependent and rise in crashes**. In a panic, "everything sells off together": ρ → 1, the cross-term stops helping, and the diversification you counted on evaporates precisely during the drawdown (2008, March 2020). Implications:
- estimate correlations with **enough data and awareness of regimes** (and shrink them — Day 122 — because sample correlation matrices are noisy and unstable),
- stress-test portfolios under **ρ → 1**,
- true diversifiers (e.g. assets that are *negatively* correlated in crises, like some tail hedges) are valuable *because* they hold up when ρ spikes.

**Hard puzzle:** a 50-stock portfolio looks well-diversified (σ_p ≈ 12% vs 20% single-name) in calm times. In a crash all pairwise correlations jump from 0.3 to 0.9. New σ_p? (`σ·√ρ̄ ≈ 0.2·√0.9 ≈ 19%` — almost the full single-name risk. The diversification was mostly an illusion of the calm regime.)

## Block 5 — Quant Thinking · the covariance matrix is the object
Generalize: for weight vector **w** and covariance matrix **Σ**,
```
σ²_p = wᵀ Σ w
```
This single quadratic form (linear algebra from the earlier module paying off) *is* portfolio risk, and it's what tomorrow's optimizer minimizes. Σ has `N(N+1)/2` parameters — for 500 stocks that's ~125,000 numbers estimated from limited noisy data, so **Σ is dangerously ill-conditioned**; practitioners use **shrinkage (Ledoit–Wolf), factor models, or PCA** to tame it. The estimation of Σ is, quietly, where most portfolio construction succeeds or fails.

**Career connection:** "why diversify?" Weak: "don't put all eggs in one basket." Strong: write `σ²_p = wᵀΣw`, show risk is sub-linear in N because of off-diagonal ρ, note the *undiversifiable* `σ√ρ̄` floor (= systematic risk → CAPM), and warn that correlations spike to 1 in crises so the matrix must be shrunk and stress-tested. That arc is a portfolio-construction interview in one breath.

## Block 6 — Python (quant) · Portfolio risk and the diversification curve
```python
import numpy as np
rng = np.random.default_rng(5)

def port_vol(w, Sigma):           # wᵀ Σ w
    return float(np.sqrt(w @ Sigma @ w))

# build Σ from equal vol (20%) and average correlation rho
def make_sigma(N, rho, vol=0.20):
    C = np.full((N, N), rho); np.fill_diagonal(C, 1.0)
    return np.outer([vol]*N, [vol]*N) * C

for N in [1, 2, 5, 20, 100]:
    S = make_sigma(N, rho=0.3)
    w = np.full(N, 1/N)
    print(f"N={N:3d}  equal-weight vol = {port_vol(w, S):.3f}")

print("undiversifiable floor σ√ρ̄ =", round(0.20*np.sqrt(0.3), 3))

# crisis: correlations jump 0.3 -> 0.9
S_calm, S_crash = make_sigma(50, 0.3), make_sigma(50, 0.9)
w = np.full(50, 1/50)
print("50-name vol  calm:", round(port_vol(w, S_calm),3),
      " crash:", round(port_vol(w, S_crash),3))
```
The vol falls toward the `≈0.11` floor as N grows — then the crash run shows it leaping back near single-name risk. The free lunch has fine print.

## Quant Thinking Track — Decomposition
Total risk decomposes into **systematic** (`σ√ρ̄`, undiversifiable, *rewarded*) + **idiosyncratic** (`~1/N`, diversifiable, *unrewarded*). This split is the conceptual seed of factor models and CAPM: the market only pays you for risk you *can't* diversify away. Hold that thought — it's literally tomorrow and Day 148.
